# Univas

A private shared digital universe for two people.

Every interaction, activity, challenge, and memory created together becomes a **star**.

A completed puzzle, a drawing, an escape room, or a shared moment is not just an activity — it becomes a permanent memory inside your universe.

---

## ✨ Core Idea

**Everything becomes a star.**

The universe is a visual timeline of your relationship.

Examples:

- 🧩 Completing a jigsaw puzzle → Puzzle Star
- 🎨 Creating artwork together → Art Star
- 🔐 Solving an escape room → Adventure Star
- 🏠 Decorating your shared room → Space Star
- 💌 Unlocking a message → Memory Star

Over time, the universe grows.

Your relationship becomes a constellation of memories.

---

# 🌠 The Universe

The main experience is a shared galaxy.

Users can explore:

- Stars
- Memories
- Activities
- Milestones
- Important moments

Each star represents something that happened.

Clicking a star opens the memory.

Example:

```

Date:
```

# 🧩 Activities

## Jigsaw Puzzle

A classic image puzzle.

Features:

- Upload image
- Split image into pieces
- Shuffle pieces
- Solve together
- Track completion

When completed:

Creates a Puzzle Star.

Stores:

- Image
- Completion percentage
- Time taken
- Difficulty
- Messages

---

# 🎨 Collaborative Drawing

A shared canvas where both people draw together.

Features:

- Real-time drawing
- Multiple tools
- Colors
- Save artwork

Examples:

- Draw each other
- Create memories
- Leave notes

When saved:

Creates an Art Star.

Stores:

- Artwork
- Date
- Creators
- Messages
